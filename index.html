すべて正規化（開始資産=1.0倍・%・枯渇率）。スマホ対応。
配分の判断は下の核心5枚＝「なぜ下げる → どう配分 → 効率で選ぶ → 総合する → 運用で守る」で完結します。

核心 5枚 配分判断の物語（なぜ → どう配分 → 効率で選ぶ → 総合する → 運用で守る）
1
なぜ・いつ危ないか — 30年の資産推移と floor（アニメ）
取り崩し中は floor(最悪5%) が年と共に沈む＝シーケンスリスク。高レバほど 0 へ。まず“下げる動機”。
→
2
どう配分するか — 脱レバ配分マップ（NASDAQ100 GP・対話）
レバナス残し%×(GP:現物金)を細かく(132マス)。相場前提・ストレス・指標(CVaR5含む)で配分を探す地図。
→
3
効率で選ぶ — 配分の効率フロンティア（成長×下振れ・対話）
各配分を点で。右上ほど良く、フロンティア＝同じ安全度で成長最大。★Claude推奨・◆codex推奨も描画。
→
4
総合する — 前提の重み × リスク回避度で最適配分（対話）
悲観/中間/楽観で結論が割れる問題の“折り合い”。前提の確率加重×γ(CRRA)で総合最適★が決まる。
→
5
運用で守る — 動的取崩ガードレール（成長×最悪時の生活水準・対話）
配分で救えない悲観を“取崩率で支出を段階削減”して受け流す。全金は金ストレスで崩落＝補正★+ガードレールが中核。
→
